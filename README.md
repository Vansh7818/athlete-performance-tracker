# Athlete Performance Tracker

An Android application developed using **Kotlin** and **Android Studio** for tracking athlete performance, workout activities, and fitness progress. The application helps users monitor daily athletic performance metrics through a clean and user-friendly mobile interface.

---

## Features

* Athlete performance tracking
* Workout and activity monitoring
* Fitness progress management
* Simple and responsive UI
* Android-based mobile application
* Structured activity flow and navigation

---

## Tech Stack

* Kotlin
* Android Studio
* XML
* Gradle
* Android SDK

---


## Installation

1. Clone the repository:

```bash
git clone https://github.com/Vansh7818/athlete-performance-tracker.git
```

2. Open the project in Android Studio

3. Sync Gradle files

4. Run the application on:

   * Android Emulator
   * Physical Android Device
